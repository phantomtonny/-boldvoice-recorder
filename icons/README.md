# アイコンについて

Chrome拡張機能には以下のサイズのアイコンが必要です：

- `icon16.png` - 16x16ピクセル
- `icon48.png` - 48x48ピクセル
- `icon128.png` - 128x128ピクセル

## アイコンの作成方法

### オンラインツールを使用する場合
1. [Canva](https://www.canva.com/)などのデザインツールでアイコンを作成
2. マイク、音声波形、録音ボタンなどのモチーフを使用
3. 各サイズでエクスポート

### 簡易的な方法
開発中は、シンプルな色付き正方形の画像でも動作します。
以下のようなオンラインツールで簡単に作成できます：
- https://www.favicon-generator.org/
- https://favicon.io/

## 注意事項
アイコンファイルがない場合、Chrome拡張機能は読み込めますが、
デフォルトのアイコンが表示されます。
